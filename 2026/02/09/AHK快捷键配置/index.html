AHK快捷键配置

Created2026-02-09|Updated2026-02-18
|Word Count:2.6k|Reading Time:9mins|Post Views:|Comments:

AHK插件下载

AutoHotkey 官网 (https://www.autohotkey.com/)

本机暂时下载了1.1版本的 AutoHotkey_1.1.37.02_setup

使用教程

此部分使用教程参考了Gemini最新的动态视图功能,作为教程进行显示 具体爬取Gemini的动态视图页面可以参考之前的博客 点击这里跳转到 Gemini 爬取教程

配置脚本

一、全局总控：Master Script (Start.ahk)

为了避免运行一堆零散的 AHK 图标，我创建了一个 Start.ahk 作为总控入口。它不仅负责引入所有子模块，还对全局环境进行了优化配置（如提升性能、强制替换旧实例等）。

二、窗口与系统快捷操作

1. 窗口一键置顶 (Always_On_Top.ahk)

按下 Ctrl + Space 即可切换当前窗口的置顶状态，并带有屏幕 ToolTip 状态提示，1.5秒后自动消失 。

2. 全局纯文本粘贴 (Paste_Plain.ahk)

使用 Ctrl + Shift + V 将剪贴板内容转换为纯文本并粘贴，自动去除网页或文档中的富文本格式 。

3. 一键唤出记事本 (open_notepad.ahk)

Alt + N 瞬间唤出记事本，用于随手记录 。

三、资源管理器 (Explorer) 增强与改造

1. 智能导航与路径跳转 (Explorer_Nav.ahk)

支持 Ctrl+左右方向键 进行历史前进后退，并为鼠标中键赋予了在搜索结果中跳转到真实目录的功能 。

2. 增强版新建文本文件 (New_Text_File.ahk)

按 Alt + Shift + N 直接在当前路径利用时间戳生成文本文档，并自动进入重命名状态 。

3. 剪贴板图片直接粘贴为文件 (paste.ahk)

免去打开画图软件的烦恼，在文件夹内按 Ctrl + V 直接将剪贴板中的图片保存为带时间戳的 PNG 文件（调用 PowerShell 实现）。

四、鼠标操作进阶 (滚动与缩放)

1. 横向滚动 (Horizontal_Move.ahk)

按住 Shift + 滚轮 实现横向滚动，适用于查看宽表格或设计图 。

(附：左键横向滚动备用方案 LButton_Horizontal.ahk，通过波浪号保留左键原生功能。部分软件内可能引起文字误选，因此在总控中默认屏蔽 。)

2. 防卡键平滑缩放 (Mouse_Zoom.ahk)

利用 右键 + 滚轮 实现平滑缩放。加入了高频定时器监控物理按键状态，彻底解决 AHK 模拟按键容易导致 Ctrl 卡死的问题 。
