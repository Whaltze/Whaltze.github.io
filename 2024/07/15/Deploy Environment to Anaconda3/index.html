包管理

安装包

conda install PackageName

安装多个包

conda install name1 name2 …

安装包并指定版本号

conda install PackageName=版本号

卸载包

conda remove PackageName

更新包

conda update PackageName

更新环境中的所有包

conda update –all

列出已安装的包

conda list

搜寻包

conda search PackageName
